Virtual Webcam Setup

To use in Zoom/Teams:

  1. Install OBS Studio
  2. Add Browser Source with this page's URL
  3. Start OBS Virtual Camera
  4. Select "OBS-Camera" in your meeting app

Phone Setup

  1. Click "Start Phone Camera"
  2. Allow camera access
  3. Share the QR code/URL with your computer
  4. Keep this page open
